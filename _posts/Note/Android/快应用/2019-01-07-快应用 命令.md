创建项目
`hap init <ProjectName>`

安装依赖
`npm install`

编译项目
手动
`npm run build`
自动(每次修改源代码文件后，都自动编译项目)
`npm run watch`

预览效果
启动HTTP服务器
`npm run server`
`npm run server -- --port 8080`


安装NodeJS,从NodeJS官网下载,6.0以上,不要使用8.0.*版本
['https://nodejs.org/en/'](https://nodejs.org/en/)

安装 hap-toolkit
`npm install -g hap-toolkit`

输出版本信息
`hap -V`
