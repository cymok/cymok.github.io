# AS 3.2 报红 Cannot resolve symbol ...

#### 1.打开 `build.gradle`

#### 2.哪个包有问题就把它的依赖先注释掉, 然后 `Sync Project`
```groovy
    //implementation 'com.android.support.constraint:constraint-layout:1.1.3'
```

#### 3.取消注释, 再 `Sync Project`
```groovy
    implementation 'com.android.support.constraint:constraint-layout:1.1.3'
```
#### 4.完成
