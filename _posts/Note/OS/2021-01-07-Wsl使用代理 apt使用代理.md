apt-get 使用 socks5 代理

1. 仅命令行一次性使用代理：
使用命令时, 加上 `-o` 参数
```
sudo apt update
```
改为
```
sudo apt -o Acquire::socks::proxy="socks5://127.0.0.1:7890/" update
```

2. 一次配置永远享受

```
sudo vim /etc/apt/apt.conf.d/12proxy
```
追加
```
Acquire::socks::proxy "socks5://127.0.0.1:1080/";
```
