协程 的 Flow

Coroutine 的 RxJava

操作符跟 RxJava 差不多，但是需要在协程内使用

```
        lifecycleScope.launch {
            flow<Int> {
                for (i in 0 until 10) {
                    emit(i) // 在每次遍历时发出事件 
                }
            }.flowOn(Dispatchers.IO)
                .onStart { // 最先开始

                }.onEach { // 每次接收

                }.catch { // 异常接收

                }.onCompletion { // 整个流完成

                }.collect {  // 数据接收

                }
        }
```
