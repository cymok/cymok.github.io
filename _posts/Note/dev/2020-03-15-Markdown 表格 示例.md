- 只需要中间的线条就可以了

```
1|2
-|-
3|4
```

1|2
-|-
3|4

大多教程都是 `---` , 其实可省略到 1 个符号 `-`

前后的 `|` 也可省略

- 对齐方式 用冒号在表头下方线条两边标注

```
左对齐|居中对齐|右对齐|默认对齐
:-|:-:|-:|-
1|2|3|4
```

左对齐|居中对齐|右对齐|默认对齐
:-|:-:|-:|-
1|2|3|4

- 换行 用 HTML 标签 `<br/>`

```
| 列1 | 列2 |
| --- | --- |
| 行1第一列<br/>行1第二列 | 行2第一列<br/>行2第二列 |
```

| 列1 | 列2 |
| --- | --- |
| 行1第一列<br/>行1第二列 | 行2第一列<br/>行2第二列 |
