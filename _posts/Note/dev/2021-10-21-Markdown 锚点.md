## 锚点

支持锚点, MD 语句
```markdown
# 标题
```

渲染后的 HTML 语句
```html
<h1 id="标题">
    标题
</h1>
```

若标题包含空格, 渲染后的 id 将会替换成 `-`

若标题包含大写字母, 渲染后的 id 将会替换成 小写字母

即 `我是 Title` 替换为 `我是-title`

## 链接

跳转到锚点的链接, MD 语句
```markdown
[跳转到标题](#标题)
```

渲染后的 HTML 语句
```html
<p>
    <a href="#标题">
        跳转到标题
    </a>
</p>
```
